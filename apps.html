PowerApps Sales Data ETL

This project demonstrates a PowerApps solution that connects to a SQL database, processes sales data, and exports the transformed data to an Excel file using Power Automate.

Steps:

  • Connect PowerApps to a SQL database.
  • Fetch sales data using SQL queries.
  • Perform ETL operations within PowerApps.
  • Export transformed data to an Excel file.

Este projeto demonstra uma solução PowerApps que se conecta a um banco de dados SQL, processa os dados de vendas e exporta os dados transformados para um arquivo Excel usando o Power Automate.

Passos:

  • Conectar o PowerApps a um banco de dados SQL.
  • Buscar dados de vendas usando consultas SQL.
  • Executar operações de ETL no PowerApps.
  • Exportar os dados transformados para um arquivo Excel.
