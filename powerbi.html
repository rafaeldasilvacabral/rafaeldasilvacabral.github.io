Power BI Project - ETL + Dashboard

Objective

Create a Power BI report that transforms raw data from an Excel file, cleans and structures the information, and presents visual insights in an interactive dashboard.

Step by Step

  • 1. Data Source: Excel file with customers, sales, and products.
  • 2. ETL in Power Query: Data import, cleaning, and transformation.
  • 3. Table Relationships: Linking customers, sales, and products.
  • 4. Dashboard Creation: KPIs, interactive charts, and detailed analysis.
  • 5. Customization: Applying theme and custom layout.

Custom Theme

The Power BI theme includes primary colors like Blue (#007ACC), Green (#4CAF50), and Gray (#6D6D6D), as well as modern fonts for a clean and professional design.
